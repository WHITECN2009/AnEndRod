## 玩法详解
| 名称      | 用法说明                                       |
|---------|--------------------------------------------|
| 末地烛     | 主手手持，对自己右键为自用；蹲下（Shift）对着其他玩家右键可为他人使用，多人互动 |
| 手铐+钥匙绑定 | 主手持手铐，副手持钥匙，按F交换（主副手调换）即可绑定，使钥匙和手铐对应       |
| 手铐      | 主手手持手铐，对着目标玩家**右键**，可为其戴上手铐                |
| 钥匙      | 主手手持钥匙，对着被手铐锁住的玩家**右键**，若钥匙匹配可解锁           |

# 命令
| 指令                               | 用途             |
|----------------------------------|----------------|
| /rodmerge 或 /rodmerge gui        | 打开小玩具兑换菜单      |
| /rodmerge reload                 | 重载配置文件         |
| /rodmerge setrodused <玩家ID> <整数> | 设置目标玩家的末地烛使用计数 |
| /rodmerge getrodused <玩家ID>      | 获取目标玩家的末地烛使用计数 |
| /rodmerge togglecuff             | 切换是否允许自身被戴上手铐  |
# 配置文件解释 （给一群臭杂鱼看的！）


```yaml
enable_pack: true #默认开启材质包，改成false就可以关掉啦
summon_slime: true #是否使用粘液末地烛的时候生成史莱姆
pack_url: "https://bgithub.xyz/WHITECN2009/AnEndRod/raw/refs/heads/master/src/main/resources/AnEndRod_Pack.zip" 
#这个材质包连接可能会失效，失效找作者去补喵！
```
